SceneAssembly Extension
====================================================================================

.. image:: /images/FE_logo_345_60.*
   :width: 345px
   :height: 60px

| |FABRIC_PRODUCT_NAME| version |FABRIC_VERSION|
| |FABRIC_COPYRIGHT|

The SceneAssembly extension defines scene abstraction interfaces,
various scene filters and processing nodes, allowing for the creation
of a procedural scene progessing graph.

See :ref:`sceneassembly` for an overview of the provided functionality.

See :ref:`SceneHub<scenehub>` for a global picture of the
:ref:`SceneGraph<scenegraph_overview>`, the :ref:`sceneassembly` and the :ref:`rtr2`.

.. warning::

  This extension is in early development stages and will grow in the next releases.
  In particular, there is currently no support for visual creation
  of processing graphs (DFG) however this is planned for future releases.
