前書と背景¶

RTVal とはデータコンテナであり、 Fabric Engine とクライアントアプリケーション間でデータの値をやり取りするために使用されます。このガイドでは RTVal がどのように動作するかについて詳しく説明し、C/C++ や Python アプリカラコレをどのように活用するかについても説明します。

この文書では、 Fabric Engine の基礎概念の理解を前提にします。もし不安があれば Core オーバービュー を参照することをお薦めします。
