FileIO エクステンション¶

Fabric Engine version 2.4.0
Copyright (c) 2010-2017 Fabric Software Inc. All rights reserved.

FileIOエクステンションは Boost::FileSystem ライブラリをラップし、KL内でいくつかの基本関数を利用できるようにします。さらに、FileIOエクステンションはバイナリデータやアスキーでの読み込みと書き込みを行うオブジェクト群とKLインターフェース群を提供します。
