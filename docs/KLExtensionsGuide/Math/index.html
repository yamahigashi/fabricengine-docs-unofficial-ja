Math エクステンション¶

Fabric Engine version 2.4.0
Copyright (c) 2010-2017 Fabric Software Inc. All rights reserved.

Mathエクステンションは Fabric Engineの至る所で使用される３D数学ライブラリを提供します。

Fabric Engine Core は基本的な数値計算法に加えて、スカラーや整数、ブール値といった基本的な低レベル数学構成群を使用できます。コアの数学についてのドキュメントは、 関数と他のグローバルな宣言 ドキュメント内で見つけることが出来ます。

三角関数はここで見つけることができます: 三角関数

丸め関数はここで見つけることができます: non_transcendental_functions_functions-functions

Math.kl ファイルは、定数とヘルパー関数の一覧を提供します。
