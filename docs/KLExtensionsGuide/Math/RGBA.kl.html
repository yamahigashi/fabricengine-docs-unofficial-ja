Types¶

RGBA (struct)¶

The RGBA type represents a 8 bit 4 channel color value, storing the components ordered as red, green, blue and alpha as Byte values.

参考

Color, ARGB

/*
** Example: RGBA
*/

RGBA red(255, 0, 0, 255);
