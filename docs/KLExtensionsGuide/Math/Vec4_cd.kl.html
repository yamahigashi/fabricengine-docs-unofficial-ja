Types¶

Vec4_cd (struct)¶

The Vec4_cd type represents a vector with 4 components. The Vec4_cd is also used as the row type for the Mat44.

注釈

Vec4_cd utilizes the KL ベクタ関数

参考

Vec2, Vec3_cd

/*
** Example: Vec4_cd
*/

Vec4_cd v(0.0, 0.0, 0.0, 0.0);
