SceneGraphToRTR Extension
====================================================================================

.. image:: /images/FE_logo_345_60.*
   :width: 345px
   :height: 60px

| |FABRIC_PRODUCT_NAME| version |FABRIC_VERSION|
| |FABRIC_COPYRIGHT|

The SceneGraphToRTR extension provides various helpers and adaptors that
allow the :ref:`SceneGraph<scenegraph_overview>` instances to be rendered with the :ref:`rtr2`.

See :ref:`scenehub_adaptors_example` and :ref:`rtr2` for an illustration of these SceneGraph to RTR
objects.

See :ref:`SceneHub<scenehub>` for a global picture of the
:ref:`SceneGraph<scenegraph_overview>`, the :ref:`sceneassembly` and the :ref:`rtr2`.
