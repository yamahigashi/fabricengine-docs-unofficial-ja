Types¶

Mat22_i (struct)¶

The Mat22_i represents a 2 by 2 matrix. It uses Vec2_i types for each of the 2 rows.

/*
** Example: Construct Mat22_i
*/

// create a matrix 2x2
Vec2_i v0(1.0, 0.0);
Vec2_i v1(0.0, 1.0);
Mat22_i mat(v0, v1);
