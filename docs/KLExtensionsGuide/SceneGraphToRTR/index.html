SceneGraphToRTR Extension¶

Fabric Engine version 2.4.0
Copyright (c) 2010-2017 Fabric Software Inc. All rights reserved.

The SceneGraphToRTR extension provides various helpers and adaptors that allow the SceneGraph instances to be rendered with the RealTime Renderer.

See SceneGraph to RTR adaptors example and RealTime Renderer for an illustration of these SceneGraph to RTR objects.

See SceneHub for a global picture of the SceneGraph, the Scene Assembly and the RealTime Renderer.
