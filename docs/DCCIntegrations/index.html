Fabric Engine DCC 統合¶

Fabric Engine version 2.4.0
Copyright (c) 2010-2017 Fabric Software Inc. All rights reserved.

初めに読んでね！¶

2.0 になりFabric Engine の全てのDCC統合は全て Canvas になりました。2.0以前までは Splice です。以前のドキュメントについてもここから同様に辿れます。

Canvas¶

Fabric は様々なDCCに (Digital Content Creation Tools)統合されています。

Canvas についてまだ馴染みが薄いのであればこちら：

  • Fabric Engine 入門
  • Canvas ユーザガイド
  • Canvas プログラマーガイド

特定のDCC統合についてのドキュメントであれば：

  • Fabric for 3ds Max
  • Fabric for Maya
  • Fabric for MODO
  • Fabric for Softimage
