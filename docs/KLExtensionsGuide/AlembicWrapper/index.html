AlembicWrapper エクステンション¶

Fabric Engine version 2.4.0
Copyright (c) 2010-2017 Fabric Software Inc. All rights reserved.

AlembicWrapper エクステンション は Alembic エクステンション への高レベルのラッピングを提供します。このレイヤーは抽象化をもたらすのですが、 AlembicIObject や AlembicOObject といったローレベルのアレンビックオブジェクトへも、これまで通りアクセス出来ます。このエクステンションの主なオブジェクトは AlembicArchiveReader と AlembicArchiveWriter です。
