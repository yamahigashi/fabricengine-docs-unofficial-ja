SceneGraph Extension¶

Fabric Engine version 2.4.0
Copyright (c) 2010-2017 Fabric Software Inc. All rights reserved.

The SceneGraph is a generic container of scene objects (SGObject), their properties (SGObjectProperty) and the object references (hierarchy).

See SceneGraph Overview for more details.

See SceneHub for a global picture of the SceneGraph, the Scene Assembly and the RealTime Renderer.

Related extensions¶

The SceneGraph extensions is closely related to these other extensions:

  • The SceneAssembly Extension defines interfaces for abstracting the scene along with nodes for filtering or processing scene graph elements.
  • The SceneGraphWrappers Extension extension defines common higher-level wrappers that define common scene types, such as SGDirectionalLight or SGCamera.
  • The SceneGraphToRTR Extension extension provides specialized adaptors from the SceneGraph to RealTime Renderer objects, such as the SGCameraToRTR.
  • The AlembicToSceneGraph Extension extension provides specialized generators mapping AlembicWrapper readers to SceneGraph objects, such as the SGAlembicGeometry.
