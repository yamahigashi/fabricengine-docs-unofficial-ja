InlineGeometryAttributeTextShape (object)¶

バージョン 2.4.0 で追加.

The InlineGeometryAttributeTextShape inherits from the InlineBaseTextShape object. It provides a shape type to draw geometry attributes as text with TextGL.

InlineBaseTextShape.
