Types¶

Vec2_c (struct)¶

The Vec2_c type represents a vector with 2 components. The Vec2_c is also used as the row type for the Mat22.

参考

Vec3, Vec4

/*
** Example: Vec2_c
*/

Vec2_c v(0.0, 0.0);
