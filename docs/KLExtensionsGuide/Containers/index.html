Containers エクステンション¶

Fabric Engine version 2.4.0
Copyright (c) 2010-2017 Fabric Software Inc. All rights reserved.

Containers エクステンション はビットマスク(bitmasks)やメモリ、異なるデータ型の数へローカライズされた配列と同様のハイパフォーマンスなインデックスセットを管理するローレベルデータコンテナ群の一連を提供します。
